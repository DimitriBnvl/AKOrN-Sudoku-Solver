# AKOrN Sudoku Solver
This repository should be used in combination with a Kaggle Notebook session specifically. 
If for whatever reason you insist on using another kind of notebook, know that the file paths take into account Kaggle's
structure.

## Purpose ##
I have recently been looking into *"ARTIFICIAL KURAMOTO OSCILLATORY NEURONS"* and wanted to recreate the paper's
results as part of a larger oscillatory neural network exploration.

With that in mind, I made this notebook to recreate the paper's results on weaker computers (in Kaggle, for example, you
are given access to two T4 GPUs).
